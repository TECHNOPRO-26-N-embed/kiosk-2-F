## 2. 要求仕様書

### 2.1 顧客の要望（ヒアリングメモ）

* 料金→4泊5日 50円 7泊8日 100円
* 延滞料金 +1日 50円
* 返却方法は営業時間内のみ
* ID発行は紙カード
* 貸出時、本の番号/コードを手入力
* 貸出状態のステータス参照
* 破損の可能性を加味して、人間が受け付けて除外処理をする
* ID発行の際、氏名・電話番号・メールアドレス・住所を登録
* 支払方法、現金・クレジットカードのみ
* CSVで出力
* 操作ログの表示

### 2.2 機能一覧

| 機能ID | 機能名             | 概要                                                         |
|--------|--------------------|--------------------------------------------------------------|
| F01    | 書籍検索           | 書籍名やコードで書籍情報を検索し、結果を表示する             |
| F02    | 貸出機能           | 利用者IDと本コードを入力し、貸出情報を登録                   |
| F03    | 延滞管理           | 返却期限超過時に延滞料金（1日50円）を自動計算                |
| F04    | 利用者ID発行       | 氏名・電話番号・メール・住所を登録し、紙カードを発行          |
| F05    | 貸出状態参照       | 書籍ごとの貸出/返却/延滞などのステータスを参照・表示         |
| F06    | 除外処理           | 破損等の理由で人間が書籍を貸出対象から除外                   |
| F07    | 支払処理           | 現金またはクレジットカードでの支払い受付                     |
| F08    | CSV出力            | 貸出・返却・利用者情報・操作ログ・書籍情報・支払等の履歴をCSV形式で出力                      |
| F09    | 操作ログ表示       | システムの操作履歴を表示                                     |
| F10    | キャンセル機能       |   現在の操作を中断し、メインメニューに戻る                                |
| F11    | 返却管理       | 返却された書籍の状態を確認し、返却処理を行う                                |
---

## 3. 制約事項

- 料金体系は「4泊5日：50円」「7泊8日：100円」とする
- 延滞料金は1日ごとに50円加算
- 返却は営業時間内のみ受付
- 利用者IDは紙カードで発行
- 本の貸出時は利用者ID/コードを手入力
- 支払方法は現金またはクレジットカードのみ
- 破損等の除外処理は人間が判断
- データ出力はCSV形式
- 操作ログを記録・表示する
- 貸出制限は一冊の本が同時に複数の利用者に同時に貸し出されない
---

## 4. 入出力仕様

### 入力

- 書籍名または書籍コード（検索・貸出時）
- 利用者ID（貸出・返却時）
- 氏名・電話番号・メールアドレス・住所（ID発行時）
- 貸出期間（4泊5日 or 7泊8日）
- 支払方法（現金/クレジットカード）
- 返却日（返却時）
- キャンセル操作（メインメニューに戻る）
- 返却書籍の状態（破損・紛失など）

### 出力

- 書籍検索結果リスト
- 貸出・返却・延滞情報
- 利用者ID（紙カード）
- 支払金額・お釣り・延滞料金
- CSVファイル（貸出・返却・利用者情報・操作ログ・書籍情報・支払履歴）
- 操作ログ一覧

---

## 5. 備考

- 破損や除外処理は人間が判断し、システムで記録する
- CSV出力やログ表示は管理者向け画面で提供
